# 🌴 Rio Tour Guide - Landing Page

Landing page desenvolvida para um guia turístico no Rio de Janeiro, com o objetivo de apresentar serviços, roteiros e facilitar o contato com clientes.

## 📸 Preview

https://site-andersonguiaturista.vercel.app/

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- (Adicione aqui se usar: Bootstrap, Tailwind, etc)

## 🎯 Funcionalidades

- Carrossel de imagens do Rio de Janeiro
- Apresentação dos serviços de turismo
- Seção de roteiros/passeios
- Botão de contato (WhatsApp)
- Layout responsivo

## 📱 Layout

O site foi desenvolvido com foco em:
- Experiência do usuário (UX)
- Design moderno e limpo
- Conversão (geração de leads)
